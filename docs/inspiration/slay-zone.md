# SlayZone (slay.zone)

**Official docs:** [https://slay.zone/docs/](https://slay.zone/docs/)

## Why it matters here

SlayZone is a **local-first** desktop app for working in real git folders: task boards, persistent terminals, embedded browser, git worktrees, and several **terminal-based AI modes** (including Claude- and Cursor-oriented flows). That aligns with how this repo thinks about **local data**, **CLI**, and **agent-assisted** development — without replacing `AGENTS.md` or our Python/CI truth.

Documentation on the site is **AI-generated** (they say so explicitly); treat it as orientation, not a legal or security guarantee.

## Relationship

The maintainer follows SlayZone’s public documentation and community as an **ecosystem inspiration** for agent + repo workflows. Product direction for **Gittan** still lives in `docs/product/` and `docs/decisions/`.
