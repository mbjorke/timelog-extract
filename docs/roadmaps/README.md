# Roadmaps

Use **`roadmaps/`** (plural) when you maintain **more than one time horizon** — for example near-term delivery vs longer product bets. One folder, multiple files; name by horizon or theme (kebab-case).

Not the same as:

- **`product/`** — vision, scope, and stable product truth; roadmaps here should **link back** to those when they set direction.
- **`ideas/`** — uncommitted exploration; promote into a roadmap row when you commit to a dated outcome.
