# Brand drafts (optional)

Put experimental PNGs here. **Nothing in this folder is used automatically.**

When a draft wins:

```bash
cp docs/brand/drafts/your-mark.png docs/brand/gittan-brand-mark.png
cp docs/brand/drafts/your-og.png docs/brand/gittan-og-card.png
./scripts/build_brand_assets.sh
```

Canonical names stay the single source of truth for the shipped favicon / README / `og-image.png`.
