# TIL by month (maintainer → agent handoff)

**TIL** = *Today I learned*. Agents document **short learnings** from the human maintainer here so the next session (and other contributors) inherit preferences without re-reading the whole thread.

- **File naming:** `YYYY-MM.md` in this directory (one file per calendar month).
- **Entry shape:** a `## YYYY-MM-DD` section per day with `- bullet` lines; multiple bullets that day are fine.
- **Policy and tone:** see **`AGENTS.md` → *Maintainer TIL (learnings from the human)***. Keep entries short; do not copy sensitive or private data from `private/`.
- **Sister doc:** `docs/ideas/team-lexicon.md` (shorthand like CR / kanin / TIL semantics).

Start the next month with a new file; older months stay in git as history.
