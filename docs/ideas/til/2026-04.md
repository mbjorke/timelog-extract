# TIL — 2026-04

*Started when we adopted daily maintainer TIL. Earlier April learnings that lived only in chat may be backfilled in short form.*

## 2026-04-18

- **Checklist beats literal counts:** Prefer asking agents for a **checklist of clear sub-steps** over hard rules like “max *N* tasks per commit” — the latter is often read as policy and mis-implemented. (See `docs/ideas/team-lexicon.md` → *TIL: checklista slår siffermiss*.)
- **Naming:** the team lexicon file is `team-lexicon.md`, not a filename that starts with `TIL-`, because **TIL** is a *kind of entry* inside the lexicon, not the name of the whole file.
- **CP** = **C**ommit + **P**ush (i den ordningen) — muntligt snabbläge i `team-lexicon`; skriv fortfarande *commit* / *push* i public PR-engelska. Andra “CP”-betydelser (engelska) **är inte** vår — disambig i lexikonet.
- **`dev`/`main` divergens:** följ `docs/runbooks/dev-main-alignment.md` (tagga gammal `dev`, föreslå **C1** reset `dev`→`main` när `main` kanonisk); moln/andra agenter: `docs/task-prompts/dev-main-alignment-handoff.md` med färdig Phase A-output.
