# Archived brand snapshots

Identical **backup copies** of the current canonical masters (same bytes as `../gittan-brand-mark.png` and `../gittan-og-card.png`). Handy if someone overwrites the canonical files by mistake.


| File                              | Role                  |
| --------------------------------- | --------------------- |
| `gittan-brand-mark-rabbit-v2.png` | Backup of square mark |
| `gittan-og-card-rabbit-v2.png`    | Backup of OG card     |


`scripts/build_brand_assets.sh` reads only the **parent folder** canonical names.
