# 2026-04 TIL

## 2026-04-25

- **GSD as workflow, not feature:** When the maintainer points at GSD for this repo, treat it first as an agent focus discipline: frame intent, plan a short checklist, execute one scope, verify with evidence, and leave the next action explicit.

## 2026-04-21

- **Review close-out is a 3-step routine, not per-comment work:** Read all CodeRabbit threads first, categorise (fix / explain / escalate), then fix in one consolidated commit, then reply to every thread — even the ones you're not fixing. Explaining "not applicable" is as important as fixing. Next time: follow the close-out checklist now in the PR template.
- **Jira-like pre-existing failures look like our problem:** CodeRabbit flagged test names that don't exist on the branch (they're in 12 newer commits on `main`). Lesson: always run the test locally first before acting on a "Critical" review comment. `python3 -m unittest <file>` takes seconds.
- **Free-form tags on project profiles beat source-derived tags** for the first-phase mobile audience: the tag meaning is about what the project is *for* the client (tech vs ops), not which tool was used — a browser session for invoice filing and one for deep research are indistinguishable at the source level.

## 2026-04-20

- If a PR branch is merged/deleted while an external coding agent is still writing, git-ref lookups can fail (`Not Found` on refs). Operationally: wait for the agent task to finish before merging/deleting the branch.
- Recovery pattern: restart from latest `origin/main`, create a fresh `task/*` branch, and re-run only the intended scoped changes.
## 2026-04-18

- **Checklist beats literal counts:** Prefer asking agents for a **checklist of clear sub-steps** over hard rules like “max *N* tasks per commit” — the latter is often read as policy and mis-implemented. (See `docs/ideas/team-lexicon.md` → *TIL: checklista slår siffermiss*.)
- **Naming:** the team lexicon file is `team-lexicon.md`, not a filename that starts with `TIL-`, because **TIL** is a *kind of entry* inside the lexicon, not the name of the whole file.
- **Grenflöde:** standard upstream är **`task/* → main`** ([`BRANCH.md`](../../../BRANCH.md)). Om en separat **`dev`**-gren finns och divergerat: `docs/runbooks/dev-main-alignment.md` (tagg, **C1** mot kanonisk `main`) + `docs/task-prompts/dev-main-alignment-handoff.md` med Phase A-output.
