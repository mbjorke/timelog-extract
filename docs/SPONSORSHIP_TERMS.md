## Single maintainer / solo consultant

One natural person with occasional help from a second person stays within section 2 of the `LICENSE` without Sponsorship, provided the second person is not a third regular user in the same period.

## Forks and internal clones

Teams that **fork or clone** the repository and run the Software internally are subject to the same **use** rules: count **people running or relying on** the tool for work, not repository copies.
