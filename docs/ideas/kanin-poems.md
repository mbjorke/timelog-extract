# Kanin poems (CodeRabbit walkthrough asides)

**Kanin** = informal name for [CodeRabbit](https://docs.coderabbit.ai/) in this repo (`docs/ideas/team-lexicon.md`). On some PRs, the bot ends its walkthrough with a short poem. They are rare enough to keep here for fun — not product truth.

**How to add:** after a PR review, copy the `## Poem` block from the CodeRabbit summary comment (or the walkthrough). One entry per PR; link the PR; use the comment date.
