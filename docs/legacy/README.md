# Legacy documentation

This folder holds **old or superseded docs** we still keep in the repo (legal/history, occasional links, “we might grep this later”). It is **not** a curated exhibit and **not** guaranteed up to date — think *attic*, not *museum*.

Active direction lives under `docs/product/`, `docs/decisions/`, `docs/specs/`, etc. See [`../README.md`](../README.md).

**Note:** `docs/brand/archive/` is separate — snapshots inside the brand tree, not this folder.
